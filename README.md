# PrivARC v3 — PrivARC Cyberpunk OS Auth
Run: `npm install && npm run dev`
Deploy: Push to GitHub → vercel.com → Vite

- Login / Signup with email
- Auto wallet generation (BIP-39)
- Wallet Connect modal: MetaMask, Rabby, WalletConnect, Coinbase, Trust, OKX, TokenPocket, Brave
- EIP-4361 sign-in flow
- Cyberpunk hexagonal background
- Boot sequence animation
